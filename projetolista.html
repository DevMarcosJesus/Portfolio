Lista de Tarefa

Descrição do Projeto

Projeto feito inteiramente com JavaScript e Framework Bootstrap, é um projeto que contém uma lista onde você pode adicionar uma tarefa/atividade e clicar em adicionar, gerando uma lista com cada item adicionado em sua ordem, caso o navegador seja fechado e reaberto a lista continua da mesma forma que foi deixada.

Além da função de adicionar temos a função remover tarefa, que remove a tarefa selecionado pelo usuário, esse projeto visa trazer uma certa organização para o usuário em geral, adicionar tarefas de estudo, trabalho ou etc.
